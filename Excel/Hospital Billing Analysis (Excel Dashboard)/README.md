Hospital Billing Analysis (Excel Dashboard)
## Project Overview

This project analyzes hospital billing data using
Microsoft Excel to uncover patterns in treatment
costs, patient demographics, medical conditions, and insurance providers.
The goal is to support data-driven healthcare decisions 
through clear insights and visual dashboards.

## Objectives

Identify medical conditions with the highest hospital billing amounts

Analyze billing variations across insurance providers

Examine how patient age groups impact total billing

Evaluate average length of hospital stay by medical condition

Compare treatment costs by gender

## Dataset Description

The dataset contains patient-level hospital records, including:

Medical Condition

Billing Amount

Insurance Provider

Patient Age

Gender

Admission Date

Discharge Date

Hospital Name
